# 老爸私房錢
此網頁能夠瀏覽、新增、修改、刪除支出紀錄。

## 功能列表
- 使用者可以瀏覽支出紀錄。
- 使用者可以新增支出紀錄。
- 使用者可以修改支出紀錄。
- 使用者可以刪除支出紀錄。
- 使用者可以在首頁可以根據支出「類別」篩選支出。

## 安裝
- 開啟終端機(Terminal)cd 到存放專案本機位置並執行: git clone https://github.com/jtuuang/expense-tracker.git
- 進入資料夾： cd expense-tracker
- 初始: npm i
- 建立資料庫: 使用 Robo 3T 建立名為 expense-tracker 的資料庫
- 載入資料: npm run seed
- 開啟程式: npm run dev

## 使用工具

- Express
